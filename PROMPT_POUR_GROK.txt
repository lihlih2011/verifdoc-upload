  - `EasyOCR` : Lecture de texte.
  - `ELA (Error Level Analysis)` : Détection de compression JPEG suspecte.
  - `Copy-Move Detection` : Algorithme maison pour détecter les copier-coller.
- **Infrastructure** : Actuellement sur Render (Free Tier), migration prévue vers OVH Public Cloud (15Go RAM).
